Group Members

Alice Reyes

Project Lead

Alice is the team lead and primary architect of MyMovieList. She designed the overall system structure, UI/UX layout inspired by AniList, and coordinated tasks among the group. She also handled API integration with TMDB and built the home page movie grid with hover previews.

Contributions

  • Project planning & task delegation
  • UI/UX design (AniList-inspired)
  • TMDB API integration
  • Home page & hover preview feature
  • Navbar & responsive layout

Ben Santos

Frontend Developer

Ben focused on the watchlist page, implementing the dynamic table rendering, filter/sort system, and the add/edit modal with TMDB search integration. He also worked on localStorage persistence and the statistics bar at the top of the list.

Contributions

  • Watchlist page (watchlist.html)
  • Dynamic table & list rendering
  • Filter, sort, and search logic
  • Add/Edit modal & form validation
  • LocalStorage data persistence

Clara Mendoza

Content & Documentation

Clara was responsible for the About the Project and Developers pages, writing all project documentation, FAQ content, and feature descriptions. She also contributed to the styling, ensuring consistency across pages and handling responsive CSS adjustments.

Contributions

  • About the Project page
  • Developers page
  • Project documentation & FAQ
  • CSS consistency & responsive fixes
  • Footer & static content

Dan Lim

JavaScript Developer

Dan handled all core JavaScript logic including the shared app.js, the search overlay, modal search against TMDB, and the score range input interaction. He also performed final testing and debugging across all pages to ensure a smooth user experience.

Contributions

  • Core app.js shared logic
  • Global search overlay
  • Score slider interaction
  • Cross-page testing & debugging
  • TMDB search in modal
